# AC SCRIPTS


## Description

This CLI tool is used for setting up AC development environment.

## Getting Started

### Dependencies

* OS: Ubuntu Linux (version >= 16.4)
* Node: >= 8.7.0

### Installing

* npm pack
* npm install -g ac-scripts-1.0.0.tgz

### Executing program

```
user@pc:~$ acScripts
```
